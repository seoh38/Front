텍스트와 관련된 input 태그들

type ="text"

- 한 줄짜리 텍스트를 입력할 수 있는 텍스트 상자이다.

type="text"에서 새로 추가된 기능

datalist : 태그를 사용하여 텍스트 필드에서 데이터 목록 중에서 값을 선택하도록 만들 수 있다. 리스트로 연결해주어야 사용이 가능하다.

type="password"

- 비밀번호를 입력할 수 있는 텍스트 상자이다.

type="search", type="url", type="email", type="tel"

- 겉 모습은 텍스트 필드와 비슷하지만 각가의 정보에 맞게 세분화된 기능을 제공하는 텍스트 상자이다.









숫자와 관련된 input 태그들

type="number"

- 입력창에 숫자를 입력하는 것이 기본이지만 브라우저에 따라서 스핀박스가 표시되기도 한다.
(스핀박스 : 입력창 오른똑에 위/아래 화살표 버튼)

type="range"

- 슬라이드 바를 통한 숫자 지정 가능



날짜/시간 관련된 input 태그

type="sate|month|week|time|datetime|datetime~local"













라디오 버튼과 체크박스 태그

type="radio"

- 여러 항목 중에 한가지만 선택해야 할 경우에 사용하는 태그이다.
- 태그 여러개를 선언하여 하나의 용도로 사용
- 라디오 버튼의 경우 관련된 버튼끼리는 name 속성이 같아야 하나만 선택이 가능하다.
- value 속성은 선택한 라디오 버튼이나 체크박스의 서버로 넘길 값을 지정한다.
- checked 속성은 기본적으로 선책할 항목이 있다면 사용한다.

남자 여자

type="checkbox"

- 여러 항목중 여러 가지 항목을 선택해야 할 경우에 사용하는 태그이다
- 체크박스의 경우 관련된 체크박스 끼리는 name 속성이 같아야 한다.
- value 속성은 선택한 라디오 버튼이나 체크박스의 서버로 넘길 값을 지정한다.
- checked 속성은 기본적으로 선책할 항목이 있다면 사용한다.

여러 값 표현 태그

select 태그와 option 태그

- 사용자가 내용을 입력하는 것이 아니고 여러 옵션 중에서 선택하게 하고 싶을때 사용하는 드롭다운 목록이다.
- 드롭다운 : 버튼을 눌렀을때 목록처럼 요소들이 보이는것



textarea 태그

- type="text" 와 비슷하지만 여러 줄의 문자를 입력할 수 있다.

progress 태그

- 작업 진행 상태를 나타내는 태그이다.
- max 값은 0보다 커야한다.
- 작업 시작을 0으로 하고 최대값에서 얼마나 진행되었는지 표시한다.

meter 태그

- 전체 크기 중에서 얼마나 차지하고 있는지 표현해준다.
- ex.하드웨어 중에서 얼마나 차지를 하고 있나??
- 현재의 값이 높은지 낮은지를 색상으로 표현해준다(row, high)
