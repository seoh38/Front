font-size

- 텍스트의 크기를 변경할 때 사용한다.

  • px 테스트
  • em 테스트
  • >% 테스트

font-weight

- 글자의 두께를 변경할 때 사용한다.

  • 원래 굵기
  • 굵은 글꼴
  • 원래 굵기보다 가늘게
  • 원래 굵기보다 더 굵게

font-variant

- 영어를 작은 대문자로 변경할 때 사용한다.

i love you

font-style

- 텍스트의 기울임을 지정할 때 사용

  • italic 테스트
  • oblique 테스트

color

- 텍스트의 색상을 지정할 때 사용한다.

  • 색상명으로 지정
  • 16진수 값으로 지정
  • rgb로 지정
  • rgba로 지정

text-decoration

- 텍스트에 줄을 긋거나 줄을 없앨때 사용한다.

  • 텍스트 아래에 줄 긋기
  • 텍스트 중간에 줄 긋기(취소선)
  • 텍스트 위에 줄 긋기
  • 하이퍼 링크 밑줄 없애기

text-transform

- 영문 텍스트의 대소문자 변환시 사용한다.

  • none
  • capitalize hello
  • uppercase
  • LOWERCASE

text-shadow

- 텍스트에 그림자 효과를 줄 때 사용한다.

HTML5 HTML5 HTML5 HTML5

letter-spacing과 word-spacing

letter-spacing

낱개 글자의 간격을 조정하고 싶을때 사용한다.

아우디코리아는 쿠페형 스포츠유틸리티차량(SUV) 'Q5 스포트백' 가솔린 모델을 출시했다고 28일 밝혔다. 출시된 차량은 '아우디 Q5 스포트백 45 TFSI 콰트로'와 '아우디 Q5 스포트백 45 TFSI 콰트로 프리미엄' 모델이다. 다음달 2일부터 판매된다. 아우디 Q5 스포트백에는 2.0L 직렬 4기통 가솔린 직분사 터보차저(TFSI) 엔진과 7단 S트로닉 자동 변속기가 탑재됐다. 최고출력 265마력, 최대토크 37.7kg.m의 동력 성능을 발휘한다. 최고 속도는 210km/h다. 제로백(정지 상태에서 시속 100km까지 도달하는 데 걸리는 시간)은 6.1초다. 아우디 고유의 사륜구동 시스템인 콰트로가 적용됐으며, 연비는 복합연비 기준 L당 10.0km다. 도심연비는 9.0km/L, 고속연비는 11.7km/L다.
